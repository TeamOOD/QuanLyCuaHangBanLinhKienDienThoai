Hướng dẫn cài đặt phần mềm quản lý Mobile Store Manager
1. Cài đặt CSDL:
Attach file QLBANHANG_LINHKIEN_DIENTHOAI.mdf trong thư mục Database vào Microsoft SQL Server 
(Yêu cầu Version 2012 hoặc cao hơn)
2. Chạy file setup.exe và làm theo hướng dẫn để cài đặt chương trình
3. Sau khi hoàn tất quá trình cài đặt, chạy file Launch QL_Ban_DienThoai.exe trên Desktop để khởi động chương trình
4. Login với tài khoản admin (Tên: admin. Mật khẩu: 1234)
